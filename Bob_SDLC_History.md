## IBM i Green Screen Connection

**IBM i (AS/400)** adalah sistem midrange IBM yang menggunakan terminal emulator 5250 untuk akses "green screen". Untuk terhubung ke IBM i, ada beberapa opsi:

### 1. **Terminal Emulator Options:**
- **IBM i Access Client Solutions (ACS)** - Official IBM tool
- **TN5250** - Open source terminal emulator
- **Mocha TN5250** - Mobile/web-based emulator
- **PuTTY** dengan TN5250 protocol

### 2. **Connection Requirements:**
- IBM i server hostname/IP address
- Port (default: 23 untuk telnet, 992 untuk SSL)
- User credentials (username & password)
- System name

### 3. **Integration Approaches:**

**A. Direct Terminal Emulation:**
```bash
# Using tn5250 (if installed)
tn5250 hostname=your-ibm-i-server
```

**B. Web-based Terminal:**
- Deploy web-based 5250 emulator
- Embed in React application using iframe

**C. API Integration:**
- Use IBM i REST APIs
- Connect via ODBC/JDBC
- Use IBM i Access APIs
